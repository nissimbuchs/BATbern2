🚀 Getting Started

All BATbern APIs use JWT authentication with AWS Cognito. Obtain your API token from the authentication service and include it in the Authorization header: Authorization: Bearer <token>

🌐 Base URLs

Production: https://api.batbern.ch/api/v1
Staging: https://api.staging.batbern.ch/api/v1
Development: http://localhost:8080/api/v1
