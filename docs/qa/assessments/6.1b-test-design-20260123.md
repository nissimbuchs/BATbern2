## Recommended Execution Order

### Phase 1: Unit Tests (Fail Fast)
1. Username generation logic (UNIT-005, -006, -007)
2. Name parsing (UNIT-008, -009)
3. Email building (UNIT-012 through -018)
4. SpeakerPool creation (UNIT-001 through -004)

### Phase 2: Integration Tests (Service Layer)
1. Company Service integration (INT-004, -005, -006)
2. Invitation flow (INT-007 through -011)
3. Batch processing (INT-013 through -017)
4. Domain events (INT-018 through -020)
5. Security/PII (INT-024, -025)

1. Single invitation (E2E-001)
2. Batch invitation (E2E-002)
3. Idempotency (E2E-003)
4. Authorization (E2E-004)
