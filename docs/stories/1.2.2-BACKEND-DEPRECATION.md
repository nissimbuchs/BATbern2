# Story 1.2.2 - Backend Files Deprecation List

## Overview

Story 1.2.2 was updated (October 2025) to use Amplify v6 direct integration instead of backend API proxy. This document lists all backend files that are now deprecated and should be removed.

**Architecture Change**:
- OLD: Frontend → Backend API Gateway → Cognito
- NEW: Frontend → Amplify v6 → Cognito (direct)
