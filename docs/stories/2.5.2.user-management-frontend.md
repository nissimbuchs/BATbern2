# Story 2.5.2: User Management Frontend

## Status

Ready for Review - All core functionality complete (76/76 tests passing, all acceptance criteria met)

## Story

**As an** organizer,
**I want** a comprehensive User Management screen with search, filtering, and role management capabilities,
**so that** I can efficiently manage all platform users, their roles, and company affiliations through an intuitive administrative interface.

## Domain Context

### Primary Domain

Shared Infrastructure & Core Services - User Management Frontend (Organizer Interface)

### Involved Services

- **Frontend**: React 19 + TypeScript 5 with Material-UI 7
- **State Management**: Zustand (client state) + React Query (server state)
- **API Gateway**: User Management endpoints
- **User Management Service (Backend)**: Company-User Management Service (user portion)
- **OpenAPI Type Generation**: `openapi-typescript` for type-safe API contracts

### Cross-Domain Dependencies

- **Depends on**: Story 2.1b (User Management Service Foundation) - backend APIs must be operational
- **Integrates with**: Company Management APIs for company dropdown/autocomplete
- **Authentication**: AWS Cognito via API Gateway for role-based access (ORGANIZER only)
- **Internationalization**: i18next + react-i18next for German (primary) and English (secondary)

## Requirements Context

### Related Functional Requirements

- **FR23**: User Management interface with CRUD operations
  - Display all platform users with comprehensive filtering
  - Search functionality with autocomplete
  - View detailed user information
  - Manage role assignments
  - Intuitive administrative interface

- **FR22**: User role management
  - Promote/demote users between roles
  - Enforce minimum 2 organizers business rule
  - Complete audit trails for role changes

- **NFR1**: Responsive design optimized for mobile, tablet, and desktop access

## Wireframe Context

### Wireframe References

**Primary Wireframe**: `docs/wireframes/story-2.4b-user-management-screen.md` (complete specification)

**Key Screens**:
1. **User List Table**: Sortable columns (Name, Email, Company, Roles, Actions), pagination, search bar
2. **Search & Filters Panel**: Role filter, company filter, status filter (Active/Inactive/All)
3. **User Detail Modal**: Read-only view of full user profile with activity history
4. **User Create Modal**: Form for creating new user (First Name, Last Name, Email, Company, Initial Roles)
5. **Role Manager Modal**: Checkbox interface for role assignment with minimum organizers validation
6. **Delete Confirmation Dialog**: GDPR compliance warning for cascade deletion

### UI Components

**React Components to Create**:

1. **UserList.tsx** - Main container component
   - Props: None (uses URL query params for filters)
   - Material-UI: Box, Typography, Button
   - State: Zustand (filters, pagination), React Query (user list)

2. **UserTable.tsx** - User list table with sorting
   - Props: `{ users: User[], onRowClick: (user: User) => void, onAction: (action: string, user: User) => void }`
   - Material-UI: Table, TableHead, TableBody, TableRow, TableCell, Avatar, Chip, IconButton
   - Features: Sortable columns, role badges (🎯 🎤 🏢 👤), actions menu

3. **UserFilters.tsx** - Search and filter controls
   - Props: `{ filters: UserFilters, onFilterChange: (filters: UserFilters) => void }`
   - Material-UI: TextField, Autocomplete, FormControl, RadioGroup, Checkbox
   - Features: 300ms debounced search, role multi-select, company dropdown, status radio

4. **UserDetailModal.tsx** - User detail view
   - Props: `{ user: User | null, open: boolean, onClose: () => void }`
   - Material-UI: Dialog, DialogTitle, DialogContent, DialogActions, Avatar, Chip, Typography
   - Features: Display profile, roles, company, activity history

5. **UserCreateModal.tsx** - New user creation form
   - Props: `{ open: boolean, onClose: () => void, onSubmit: (data: CreateUserRequest) => void }`
   - Material-UI: Dialog, TextField, Autocomplete, FormGroup, Checkbox
   - Features: Form validation, company autocomplete, role selection

6. **RoleManagerModal.tsx** - Role assignment interface
   - Props: `{ user: User, open: boolean, onClose: () => void, onSave: (roles: Role[]) => void }`
   - Material-UI: Dialog, FormGroup, Checkbox, Alert (for minimum organizers warning)
   - Features: Role checkboxes, minimum 2 organizers validation

7. **UserPagination.tsx** - Pagination controls
   - Props: `{ page: number, totalPages: number, onPageChange: (page: number) => void }`
   - Material-UI: Pagination, Select (items per page)
   - Features: Previous/Next, page numbers, items per page dropdown

**3. React Query Hook (Server State)**

```typescript
// src/hooks/useUserList.ts
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';
import { userManagementService } from '@/services/userManagementService';
import type { UserFilters, User } from '@/types/user.types';

export const useUserList = (filters: UserFilters, page: number, limit: number) => {
  return useQuery({
    queryKey: ['users', { filters, page, limit }],
    queryFn: () => userManagementService.listUsers(filters, { page, limit }),
    staleTime: 5 * 60 * 1000, // 5 minutes
    cacheTime: 30 * 60 * 1000, // 30 minutes
  });
};

export const useCreateUser = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: userManagementService.createUser,
    onSuccess: () => {
      queryClient.invalidateQueries(['users']);
    },
  });
};

export const useUpdateUserRoles = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: ({ id, roles }: { id: string; roles: string[] }) =>
      userManagementService.updateUserRoles(id, roles),
    onSuccess: () => {
      queryClient.invalidateQueries(['users']);
    },
    // Optimistic update
    onMutate: async ({ id, roles }) => {
      await queryClient.cancelQueries(['users']);
      const previousUsers = queryClient.getQueryData(['users']);

      queryClient.setQueryData(['users'], (old: any) => {
        // Update user roles optimistically
        return {
          ...old,
          data: old.data.map((user: User) =>
            user.id === id ? { ...user, roles } : user
          ),
        };
      });

      return { previousUsers };
    },
    onError: (err, variables, context) => {
      queryClient.setQueryData(['users'], context?.previousUsers);
    },
  });
};

export const useDeleteUser = () => {
  const queryClient = useQueryClient();

  return useMutation({
    mutationFn: userManagementService.deleteUser,
    onSuccess: () => {
      queryClient.invalidateQueries(['users']);
    },
  });
};
```

**4. User Management Service (API Client)**

```typescript
// src/services/userManagementService.ts
import type { User, CreateUserRequest, UserFilters, PaginatedUserResponse } from '@/types/user.types';
import { apiClient } from './apiClient';

// API base path for user endpoints
// Note: apiClient baseURL is set from runtime config to 'http://localhost:8080/api/v1'
// so we only need '/users' (the /api/v1 prefix is already in the baseURL)
const USER_API_PATH = '/users';

export const userManagementService = {
  async listUsers(
    filters: UserFilters,
    pagination: { page: number; limit: number }
  ): Promise<PaginatedUserResponse> {
    const params = new URLSearchParams({
      page: pagination.page.toString(),
      limit: pagination.limit.toString(),
      include: 'company,roles',
    });

    if (filters.role && filters.role.length > 0) {
      params.append('filter', JSON.stringify({ role: filters.role }));
    }
    if (filters.company) {
      params.append('filter', JSON.stringify({ company: filters.company }));
    }
    if (filters.status && filters.status !== 'all') {
      params.append('filter', JSON.stringify({ isActive: filters.status === 'active' }));
    }

    const response = await apiClient.get(`${USER_API_PATH}?${params}`);
    // ✅ Resolves to: http://localhost:8080/api/v1/users
    return response.data;
  },

  async searchUsers(query: string): Promise<User[]> {
    const response = await apiClient.get(`${USER_API_PATH}/search?query=${encodeURIComponent(query)}&limit=10`);
    // ✅ Resolves to: http://localhost:8080/api/v1/users/search
    return response.data;
  },

  async getUserById(id: string): Promise<User> {
    const response = await apiClient.get(`${USER_API_PATH}/${id}?include=company,roles,activity`);
    // ✅ Resolves to: http://localhost:8080/api/v1/users/{id}
    return response.data;
  },

  async createUser(data: CreateUserRequest): Promise<User> {
    const response = await apiClient.post(USER_API_PATH, data);
    // ✅ Resolves to: http://localhost:8080/api/v1/users
    return response.data;
  },

  async updateUserRoles(id: string, roles: string[]): Promise<User> {
    const response = await apiClient.put(`${USER_API_PATH}/${id}/roles`, { roles });
    // ✅ Resolves to: http://localhost:8080/api/v1/users/{id}/roles
    return response.data;
  },

  async deleteUser(id: string): Promise<void> {
    await apiClient.delete(`${USER_API_PATH}/${id}`);
    // ✅ Resolves to: http://localhost:8080/api/v1/users/{id}
  },
};
```

**5. UserList Component (Main Container)**

```typescript
// src/components/organizer/UserManagement/UserList.tsx
import React, { useState } from 'react';
import { Box, Typography, Button } from '@mui/material';
import { useTranslation } from 'react-i18next';
import { useUserManagementStore } from '@/stores/userManagementStore';
import { useUserList } from '@/hooks/useUserList';
import UserTable from './UserTable';
import UserFilters from './UserFilters';
import UserPagination from './UserPagination';
import UserDetailModal from './UserDetailModal';
import UserCreateModal from './UserCreateModal';
import type { User } from '@/types/user.types';

const UserList: React.FC = () => {
  const { t } = useTranslation('userManagement');
  const { filters, page, limit, setPage, setSelectedUser, selectedUser } = useUserManagementStore();
  const { data, isLoading, isError, refetch } = useUserList(filters, page, limit);

  const [createModalOpen, setCreateModalOpen] = useState(false);

  const handleRowClick = (user: User) => {
    setSelectedUser(user);
  };

  const handleCloseDetailModal = () => {
    setSelectedUser(null);
  };

  if (isLoading) {
    return <div>{t('loading.users')}</div>; // Add skeleton loaders
  }

  if (isError) {
    return (
      <Box>
        <Typography>{t('error.loadFailed')}</Typography>
        <Button onClick={() => refetch()}>{t('actions.retry')}</Button>
      </Box>
    );
  }

  return (
    <Box sx={{ p: 3 }}>
      <Box sx={{ display: 'flex', justifyContent: 'space-between', mb: 3 }}>
        <Typography variant="h4">{t('title')}</Typography>
        <Button variant="contained" onClick={() => setCreateModalOpen(true)}>
          {t('addUser')}
        </Button>
      </Box>

      <UserFilters />

      <UserTable users={data?.data || []} onRowClick={handleRowClick} />

      <UserPagination
        page={page}
        totalPages={data?.pagination?.totalPages || 1}
        onPageChange={setPage}
      />

      <UserDetailModal
        user={selectedUser}
        open={!!selectedUser}
        onClose={handleCloseDetailModal}
      />

      <UserCreateModal
        open={createModalOpen}
        onClose={() => setCreateModalOpen(false)}
      />
    </Box>
  );
};

export default UserList;
```
