## Dev Notes - Implementation Guide

### Task 1: Fix robots.txt Serving

**Problem:** CloudFront is routing `/robots.txt` through SPA, returning HTML instead of the text file.

**Solution 1: S3 Upload (Recommended)**
```bash
# Upload robots.txt directly to S3 root
aws s3 cp web-frontend/public/robots.txt s3://staging-batbern-frontend/robots.txt \
  --content-type "text/plain" \
  --cache-control "public, max-age=3600" \
  --profile batbern-staging
```

**Solution 2: CloudFront Behavior (Alternative)**
Create a new CloudFront behavior specifically for `/robots.txt`:
```typescript
// infrastructure/lib/frontend-stack.ts
distribution.addBehavior('/robots.txt', origin, {
  viewerProtocolPolicy: cloudfront.ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
  cachePolicy: cloudfront.CachePolicy.CACHING_OPTIMIZED,
  responseHeadersPolicy: cloudfront.ResponseHeadersPolicy.CORS_ALLOW_ALL_ORIGINS,
});
```

**Verification:**
```bash
# Should return text/plain, not text/html
curl -I https://staging.batbern.ch/robots.txt

# Should show robots.txt content
curl https://staging.batbern.ch/robots.txt
```

### Task 2: Code Splitting for MUI

**Current Issue:** MUI bundle is 118KB, but 72% (85KB) is unused on initial load.

**Solution: Lazy Load Heavy Components**

```typescript
// src/pages/organizer/index.tsx
import { lazy } from 'react';

// Lazy load organizer routes (includes heavy MUI components)
const OrganizerDashboard = lazy(() => import('./OrganizerDashboard'));
const EventManagement = lazy(() => import('./EventManagement'));
const SpeakerManagement = lazy(() => import('./SpeakerManagement'));

// Wrap with Suspense
import { Suspense } from 'react';
import { Loader2 } from 'lucide-react';

const LoadingFallback = () => (
  <div className="flex justify-center items-center min-h-screen">
    <Loader2 className="h-12 w-12 animate-spin text-blue-400" />
  </div>
);

export const OrganizerRoutes = () => (
  <Suspense fallback={<LoadingFallback />}>
    <Routes>
      <Route path="dashboard" element={<OrganizerDashboard />} />
      <Route path="events" element={<EventManagement />} />
      <Route path="speakers" element={<SpeakerManagement />} />
    </Routes>
  </Suspense>
);
```

**Update vite.config.ts:**
```typescript
// vite.config.ts - manualChunks
manualChunks: {
  vendor: ['react', 'react-dom'],
  'mui-core': ['@mui/material/Button', '@mui/material/TextField', '@mui/material/Typography'],
  'mui-heavy': ['@mui/material/Autocomplete', '@mui/material/Tabs', '@mui/material/Tooltip'],
  aws: ['aws-amplify', '@aws-amplify/ui-react'],
  query: ['@tanstack/react-query'],
  router: ['react-router-dom'],
  i18n: ['react-i18next', 'i18next', 'i18next-browser-languagedetector'],
}
```

### Task 3: Optimize Images for LCP

**Add Explicit Dimensions:**
```tsx
// Before (causes CLS)
<img src="/hero-image.jpg" alt="Event" />

// After (prevents CLS)
<img
  src="/hero-image.jpg"
  alt="Event"
  width={1200}
  height={630}
  loading="lazy"  // Only for below-fold images
  fetchpriority="high"  // Only for LCP image
/>
```

**Convert to WebP:**
```bash
# Install sharp for image optimization
npm install --save-dev sharp

# Create optimization script
node scripts/optimize-images.js
```

**Preload LCP Image:**
```html
<!-- public/index.html -->
<head>
  <!-- Preload hero image (LCP element) -->
  <link rel="preload" as="image" href="/hero-image.webp" fetchpriority="high">
</head>
```

### Task 4: Fix Cumulative Layout Shift

**Reserve Space for Dynamic Content:**
```tsx
// HeroSection.tsx
<div className="relative" style={{ aspectRatio: '16/9' }}>
  {event?.imageUrl ? (
    <img
      src={event.imageUrl}
      alt={event.title}
      width={1200}
      height={675}
      className="w-full h-full object-cover"
    />
  ) : (
    <div className="w-full h-full bg-gray-800" /> // Skeleton
  )}

// VenueMap.tsx - Reserve space for iframe
<div style={{ height: '400px' }}>
  <iframe
    src={mapUrl}
    width="100%"
    height="400"
    style={{ border: 0 }}
    loading="lazy"
  />
</div>
```

**Font Loading Strategy:**
```html
<!-- public/index.html -->
<link
  rel="preload"
  href="/fonts/inter-v12-latin-regular.woff2"
  as="font"
  type="font/woff2"
  crossorigin
>

<style>
  @font-face {
    font-family: 'Inter';
    font-style: normal;
    font-weight: 400;
    font-display: swap; /* Prevents invisible text, allows layout shift */
    src: url('/fonts/inter-v12-latin-regular.woff2') format('woff2');
  }
</style>
```

### Task 6: CloudFront Caching Configuration

**Update CDK Stack:**
```typescript
// infrastructure/lib/frontend-stack.ts
const distribution = new cloudfront.Distribution(this, 'Distribution', {
  defaultBehavior: {
    origin: s3Origin,
    viewerProtocolPolicy: cloudfront.ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
    cachePolicy: new cloudfront.CachePolicy(this, 'CachePolicy', {
      cachePolicyName: 'BATbern-SPA-Policy',
      defaultTtl: Duration.hours(1),  // index.html
      maxTtl: Duration.days(365),
      minTtl: Duration.seconds(0),
      enableAcceptEncodingBrotli: true,
      enableAcceptEncodingGzip: true,
    }),
  },
  additionalBehaviors: {
    '/assets/*': {
      origin: s3Origin,
      viewerProtocolPolicy: cloudfront.ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
      cachePolicy: cloudfront.CachePolicy.CACHING_OPTIMIZED, // 1 year
      compress: true,
    },
    '/robots.txt': {
      origin: s3Origin,
      viewerProtocolPolicy: cloudfront.ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
      cachePolicy: new cloudfront.CachePolicy(this, 'RobotsCachePolicy', {
        cachePolicyName: 'BATbern-Robots-Policy',
        defaultTtl: Duration.hours(1),
        maxTtl: Duration.hours(24),
        minTtl: Duration.seconds(0),
      }),
    },
    '/sitemap.xml': {
      origin: s3Origin,
      viewerProtocolPolicy: cloudfront.ViewerProtocolPolicy.REDIRECT_TO_HTTPS,
      cachePolicy: new cloudfront.CachePolicy(this, 'SitemapCachePolicy', {
        cachePolicyName: 'BATbern-Sitemap-Policy',
        defaultTtl: Duration.hours(1),
        maxTtl: Duration.hours(24),
        minTtl: Duration.seconds(0),
      }),
    },
  },
});
```
