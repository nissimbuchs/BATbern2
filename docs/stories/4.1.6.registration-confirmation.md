# Story 4.1.6: Registration Confirmation with QR Code

## Dependencies
**Depends on**: Story 4.1.5a (Registration Architecture Foundation)

## Status
Approved

## Story
**As a** newly registered attendee (anonymous or authenticated),
**I want** to receive a confirmation with QR code and calendar export,
**so that** I have proof of registration and can add the event to my calendar.

**Note**: Anonymous registrations are automatically linked to user account when they later create a Cognito account with the same email (per ADR-005).

## Domain Context

### Primary Domain
Attendee Experience Domain (Registration confirmation)

### Involved Services
- web-frontend (React 19 + TypeScript)
- Event Management Service (via API Gateway)
- API Gateway: `GET /api/v1/events/{eventCode}/registrations/{confirmationCode}`, `GET /api/v1/events/{eventCode}/registrations/{confirmationCode}/qr`

### Cross-Domain Dependencies
- Email service sends confirmation email with QR code (async)
- Anonymous registration support per ADR-005
- Confirmation code format: `BAT-YYYY-NNNNNN` (e.g., "BAT-2025-000123")

## Acceptance Criteria

1. `/registration-confirmation/:confirmationCode` displays confirmation page (public access)
2. **API Integration**: Fetch registration details via `GET /api/v1/events/{eventCode}/registrations/{confirmationCode}`
3. Success message with confirmation code (format: `BAT-YYYY-NNNNNN`)
4. QR code displayed from `GET /api/v1/events/{eventCode}/registrations/{confirmationCode}/qr`
5. Calendar export button generates .ics file (no auth required)
6. Email confirmation already sent by backend (automatic, no action needed)
7. Social sharing buttons (LinkedIn, Twitter/X, Email)
8. "Register Another Person" link returns to `/register/{eventCode}`
9. Confetti animation on page load (celebration effect)
10. Dietary preferences update section (optional, can update after registration)
11. Account linking info: "Create an account to manage your registrations" (for anonymous users)

## Dev Notes - Implementation Guide

### Backend API Implementation

**RegistrationController.java** (add QR endpoint):
```java
/**
 * Generate QR code for registration (PUBLIC ACCESS)
 * Anyone with registration ID can get their QR code
 */
@GetMapping("/registrations/{registrationId}/qr")
public ResponseEntity<byte[]> getRegistrationQR(
    @PathVariable String registrationId
) throws IOException {
    Registration registration = registrationService.getRegistrationById(registrationId);

    // Generate QR code with registration data (JSON format)
    String qrData = String.format(
        "{\"registrationId\":\"%s\",\"eventId\":\"%s\",\"attendeeName\":\"%s\"}",
        registration.getId(),
        registration.getEvent().getId(),
        registration.getFirstName() + " " + registration.getLastName()
    );

    byte[] qrCodeImage = qrCodeService.generateQRCode(qrData, 300, 300);

    HttpHeaders headers = new HttpHeaders();
    headers.setContentType(MediaType.IMAGE_PNG);
    headers.setContentLength(qrCodeImage.length);

    return ResponseEntity.ok()
        .headers(headers)
        .body(qrCodeImage);
}
```

**QRCodeService.java** (using ZXing):
```java
@Service
public class QRCodeService {
    public byte[] generateQRCode(String data, int width, int height) throws IOException {
        QRCodeWriter qrCodeWriter = new QRCodeWriter();
        BitMatrix bitMatrix = qrCodeWriter.encode(
            data,
            BarcodeFormat.QR_CODE,
            width,
            height
        );

        ByteArrayOutputStream pngOutputStream = new ByteArrayOutputStream();
        MatrixToImageWriter.writeToStream(bitMatrix, "PNG", pngOutputStream);
        return pngOutputStream.toByteArray();
    }
}
```

### Frontend API Integration

**Extend eventApiClient.ts:**
```typescript
// Add to web-frontend/src/services/eventApiClient.ts

/**
 * Get registration details (public access with registration ID)
 */
async getRegistration(registrationId: string): Promise<Registration> {
  try {
    const response = await apiClient.get<Registration>(
      `/registrations/${registrationId}`
    );
    return response.data;
  } catch (error) {
    throw this.transformError(error);
  }
}

/**
 * Get QR code for registration (public access)
 * Returns a blob URL that can be used in <img src={url} />
 */
async getRegistrationQR(registrationId: string): Promise<string> {
  try {
    const response = await apiClient.get(`/registrations/${registrationId}/qr`, {
      responseType: 'blob',
    });
    return URL.createObjectURL(response.data);
  } catch (error) {
    throw this.transformError(error);
  }
}
```

**Registration interface (extend in event.types.ts if needed):**
```typescript
export interface Registration {
  id: string;
  registrationNumber: string;
  eventId: string;
  eventTitle: string;
  attendeeId: string;
  attendeeName: string;
  attendeeEmail: string;
  status: 'CONFIRMED' | 'PENDING' | 'CANCELLED';
  selectedSessions: Array<{
    id: string;
    title: string;
    startTime: string;
    endTime: string;
    room: string;
  }>;
  createdAt: string;
}
```

### Component Implementation

**RegistrationConfirmationPage.tsx:**
```typescript
import { useEffect, useState } from 'react';
import { useParams, Link } from 'react-router-dom';
import { useQuery } from '@tanstack/react-query';
import confetti from 'canvas-confetti';
import { createEvent } from 'ics';
import { PublicLayout } from '@/components/public/PublicLayout';
import { Button } from '@/components/public/ui/button';
import { Card } from '@/components/public/ui/card';
import { CheckCircle2, Calendar, Download, ArrowLeft } from 'lucide-react';
import { eventApiClient } from '@/services/eventApiClient';

export const RegistrationConfirmationPage = () => {
  const { registrationId } = useParams();
  const [qrCodeUrl, setQrCodeUrl] = useState<string | null>(null);

  const { data: registration, isLoading } = useQuery({
    queryKey: ['registrations', registrationId],
    queryFn: () => eventApiClient.getRegistration(registrationId!),
    enabled: !!registrationId,
  });

  // Confetti on mount
  useEffect(() => {
    if (registration) {
      confetti({
        particleCount: 100,
        spread: 70,
        origin: { y: 0.6 },
      });
    }
  }, [registration]);

  // Fetch QR code
  useEffect(() => {
    if (registrationId) {
      eventApiClient.getRegistrationQR(registrationId).then(setQrCodeUrl);
    }
  }, [registrationId]);

  const handleDownloadCalendar = () => {
    if (!registration) return;

    const event = {
      start: [2025, 3, 15, 18, 0], // Parse from registration
      duration: { hours: 3 },
      title: registration.eventTitle,
      description: `Registration #${registration.registrationNumber}`,
      location: 'Bern TechHub',
      status: 'CONFIRMED',
      busyStatus: 'BUSY',
    };

    createEvent(event, (error, value) => {
      if (error) {
        console.error(error);
        return;
      }
      const blob = new Blob([value], { type: 'text/calendar' });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = `batbern-registration-${registration.registrationNumber}.ics`;
      a.click();
      URL.revokeObjectURL(url);
    });
  };

  if (isLoading) {
    return (
      <PublicLayout>
        <div className="container mx-auto px-4 py-24 text-center">
          <p>Loading confirmation...</p>
        </div>
      </PublicLayout>
    );
  }

  if (!registration) {
    return (
      <PublicLayout>
        <div className="container mx-auto px-4 py-24 text-center">
          <p>Registration not found</p>
        </div>
      </PublicLayout>
    );
  }

  return (
    <PublicLayout>
      <div className="container mx-auto px-4 py-12 max-w-3xl">
        {/* Success Header */}
        <div className="text-center mb-8">
          <CheckCircle2 className="h-16 w-16 text-green-400 mx-auto mb-4" />
          <h1 className="text-4xl font-light mb-2">Registration Confirmed!</h1>
          <p className="text-xl text-zinc-400">
            Registration #{registration.registrationNumber}
          </p>
        </div>

        {/* QR Code */}
        {qrCodeUrl && (
          <Card className="p-8 mb-8 text-center">
            <h2 className="text-xl font-light mb-4">Your Check-In QR Code</h2>
            <div className="inline-block p-4 bg-white rounded-lg">
              <img src={qrCodeUrl} alt="Registration QR Code" className="w-48 h-48" />
            </div>
            <p className="mt-4 text-sm text-zinc-400">
              Present this QR code at the event for quick check-in
            </p>
          </Card>
        )}

        {/* Registration Details */}
        <Card className="p-6 mb-8">
          <h2 className="text-xl font-light mb-4">Registration Details</h2>
          <div className="space-y-2 text-sm">
            <p><span className="text-zinc-400">Name:</span> {registration.attendeeName}</p>
            <p><span className="text-zinc-400">Email:</span> {registration.attendeeEmail}</p>
            <p><span className="text-zinc-400">Event:</span> {registration.eventTitle}</p>
          </div>

          <div className="mt-6">
            <h3 className="font-light mb-2">Selected Sessions ({registration.selectedSessions.length})</h3>
            <ul className="space-y-2">
              {registration.selectedSessions.map((session) => (
                <li key={session.id} className="text-sm text-zinc-400">
                  {session.startTime} - {session.endTime}: {session.title} ({session.room})
                </li>
              ))}
            </ul>
          </div>
        </Card>

        {/* Actions */}
        <div className="flex flex-col gap-4">
          <Button onClick={handleDownloadCalendar} size="lg" className="w-full">
            <Calendar className="h-4 w-4 mr-2" />
            Add to Calendar
          </Button>

          <div className="grid grid-cols-2 gap-4">
            <Button asChild variant="outline">
              <Link to="/current-event">
                <ArrowLeft className="h-4 w-4 mr-2" />
                Back to Event
              </Link>
            </Button>
            <Button asChild variant="outline">
              <Link to="/attendee/dashboard">
                View My Registrations
              </Link>
            </Button>
          </div>
        </div>

        {/* Confirmation Email Notice */}
        <p className="mt-8 text-center text-sm text-zinc-400">
          A confirmation email has been sent to {registration.attendeeEmail}
        </p>
      </div>
    </PublicLayout>
  );
};
```
