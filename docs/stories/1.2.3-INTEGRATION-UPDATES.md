**File Checked**: `web-frontend/src/services/auth/authService.ts`

**Findings**:

**CRITICAL FINDING**: Story's code example used raw Cognito SDK which is WRONG!

**Original Story Code (INCORRECT)**:
```typescript
import { CognitoIdentityProviderClient, SignUpCommand } from "@aws-sdk/client-cognito-identity-provider";

const cognitoClient = new CognitoIdentityProviderClient({
  region: process.env.REACT_APP_AWS_REGION,
});
```

**Correct Pattern (UPDATED)**:
```typescript
import { signUp } from '@/services/auth/authService';  // Use existing Amplify v6 service

export const useRegistration = () => {
  return useMutation({
    mutationFn: async (data: RegistrationFormData) => {
      const result = await signUp({
        username: data.email,
        password: data.password,
        attributes: {
          email: data.email,
          given_name: data.firstName,
          family_name: data.lastName,
          'custom:language': i18n.language,
          'custom:newsletter_optin': data.newsletterOptIn.toString(),
        },
      });
      return result;
    },
  });
};
```

**Required Service Updates**:

### 2. Corrected useRegistration Hook Code Example

**Before**:
```typescript
import { CognitoIdentityProviderClient, SignUpCommand } from "@aws-sdk/client-cognito-identity-provider";

const cognitoClient = new CognitoIdentityProviderClient({
  region: process.env.REACT_APP_AWS_REGION, // WRONG!
});

await cognitoClient.send(new SignUpCommand({...})); // WRONG!
```

**After**:
```typescript
import { signUp } from '@/services/auth/authService';  // Use existing Amplify v6 service
import { useTranslation } from 'react-i18next';

export const useRegistration = () => {
  const { i18n } = useTranslation();

  return useMutation({
    mutationFn: async (data: RegistrationFormData) => {
      const result = await signUp({
        username: data.email,
        password: data.password,
        attributes: {
          email: data.email,
          given_name: data.firstName,
          family_name: data.lastName,
          'custom:language': i18n.language,
          'custom:newsletter_optin': data.newsletterOptIn.toString(),
        },
      });

      if (!result.isSignUpComplete) {
        // Email verification required
        return { requiresVerification: true, email: data.email };
      }

      return result;
    },
    onError: (error: Error) => {
      console.error('Registration failed:', error);
      throw error;
    },
  });
};
```
