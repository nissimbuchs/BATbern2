# Story 4.1.5: Registration Wizard (2-Step Flow)

## Dependencies
**Depends on**: Story 4.1.5a (Registration Architecture Foundation)

## Status
Done

## Story
**As a** public website visitor,
**I want** to register for the current event through an intuitive 2-step wizard,
**so that** I can secure my spot at the event with minimal friction.

## Implementation Approach: Hybrid Registration UX

This story implements a **hybrid registration approach** combining the best of both inline and dedicated page experiences:

### Primary UX: Inline Expandable Registration (Accordion-Style)
- **Location**: Directly within homepage HeroSection
- **Trigger**: Clicking "Register Now" button expands registration form inline
- **Benefits**: Reduces friction, keeps user on homepage, seamless UX
- **Behavior**:
  - Step 1 expands within hero section
  - Accordion-style transitions: Step 1 collapses when Step 2 expands
  - After submission, redirect to `/registration-confirmation/:confirmationCode`

### Secondary UX: Dedicated Registration Page
- **Route**: `/register/:eventCode`
- **Use Cases**: Shareable links, email CTAs, social media posts, direct bookmarking
- **Benefits**: Dedicated URL for marketing, deep linking support
- **Implementation**: **Same wizard components** as inline (DRY principle)

### Component Reuse
All registration wizard components work in **both contexts** with no duplication:
- `RegistrationWizard.tsx` - Shared wizard container
- `RegistrationAccordion.tsx` - Shared accordion wrapper
- `PersonalDetailsStep.tsx` - Shared Step 1 component
- `ConfirmRegistrationStep.tsx` - Shared Step 2 component

See `/docs/wireframes/story-4.1.5-inline-registration-accordion.md` for detailed visual specifications of both inline and dedicated page flows.

## Acceptance Criteria

1. **Inline Registration**: Clicking "Register Now" on homepage expands registration wizard within HeroSection
2. **Dedicated Page**: `/register/:eventCode` route displays RegistrationWizard (anonymous access, no auth required)
3. **Component Reuse**: Same wizard components work in both inline and dedicated contexts
4. **Step 1**: Personal details form with validation
   - Fields: firstName, lastName, email, company, role
   - Validation: required fields, email format
   - react-hook-form + zod validation
5. **Accordion Transitions**: Step 1 collapses when Step 2 expands (smooth 300ms animation)
6. **Step 2**: Review & confirm
   - Display summary of personal info (read-only card)
   - Communication preferences (newsletter, event reminders)
   - Special requests (dietary, accessibility)
   - Terms & conditions checkbox (required)
   - "Complete Registration" button
7. Progress indicator shows current step (1/2, 2/2)
8. Navigation: [Back], [Next], [Cancel] buttons with proper validation
9. Form state preserved when navigating between steps
10. **API Integration**: `POST /api/v1/events/{eventCode}/registrations` on final submit
11. **Event-level registration**: User registers for entire event (not individual sessions)
12. Validation errors displayed inline with clear messages
13. On success: Redirect to confirmation page with confirmationCode (format: BAT-YYYY-NNNNNN)
14. **Inline-specific**: HeroSection expands smoothly (500ms transition) when "Register Now" clicked
15. **Inline-specific**: After successful registration, hero collapses back to default state before redirect

## Dev Notes - Implementation Guide

### Backend API Implementation

**RegistrationController.java** (Event Management Service):
```java
/**
 * Create registration (PUBLIC ACCESS - no auth required)
 * Allows anonymous public users to register for events
 */
@PostMapping("/events/{eventCode}/registrations")
public ResponseEntity<RegistrationDTO> createRegistration(
    @PathVariable String eventCode,
    @RequestBody @Valid CreateRegistrationRequest request
) {
    Registration registration = registrationService.createAnonymousRegistration(eventCode, request);
    RegistrationDTO dto = registrationMapper.toDTO(registration);
    return ResponseEntity.status(HttpStatus.CREATED).body(dto);
}

/**
 * Get registration by ID (PUBLIC ACCESS)
 * Anyone with registration ID can view their registration
 */
@GetMapping("/events/{eventCode}/registrations/{confirmationCode}")
public ResponseEntity<RegistrationDTO> getRegistration(
    @PathVariable String eventCode,
    @PathVariable String confirmationCode
) {
    Registration registration = registrationService.getByConfirmationCode(eventCode, confirmationCode);
    return ResponseEntity.ok(registrationMapper.toDTO(registration));
}
```

```typescript
// Add to web-frontend/src/services/eventApiClient.ts

/**
 * Create event registration (public access - anonymous allowed)
 * Per ADR-005: Anonymous registration with email-based account linking
 */
async createRegistration(
  eventCode: string,
  data: RegistrationRequest
): Promise<Registration> {
  try {
    const response = await apiClient.post<Registration>(
      `/events/${eventCode}/registrations`,
      data
    );
    return response.data;
  } catch (error) {
    throw this.transformError(error);
  }
}

/**
 * Get registration by confirmation code (public access)
 */
async getRegistration(
  eventCode: string,
  confirmationCode: string
): Promise<Registration> {
  try {
    const response = await apiClient.get<Registration>(
      `/events/${eventCode}/registrations/${confirmationCode}`
    );
    return response.data;
  } catch (error) {
    throw this.transformError(error);
  }
}
```

### Component Implementation

**RegistrationWizard.tsx (2-Step Flow):**
```typescript
import { useState } from 'react';
import { useParams, useNavigate } from 'react-router-dom';
import { PublicLayout } from '@/components/public/PublicLayout';
import { PersonalDetailsStep } from '@/components/public/Registration/PersonalDetailsStep';
import { ConfirmRegistrationStep } from '@/components/public/Registration/ConfirmRegistrationStep';
import { Button } from '@/components/public/ui/button';
import { eventApiClient } from '@/services/eventApiClient';

export const RegistrationWizard = () => {
  const { eventCode } = useParams();
  const navigate = useNavigate();
  const [currentStep, setCurrentStep] = useState(1);
  const [formData, setFormData] = useState({
    firstName: '',
    lastName: '',
    email: '',
    company: '',
    role: '',
    termsAccepted: false,
    communicationPreferences: {
      newsletterSubscribed: false,
      eventReminders: true,
    },
    specialRequests: '',
  });
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleNext = () => setCurrentStep((prev) => Math.min(prev + 1, 2));
  const handleBack = () => setCurrentStep((prev) => Math.max(prev - 1, 1));
  const handleCancel = () => {
    if (confirm('Are you sure you want to cancel registration?')) {
      navigate(`/events/${eventCode}`);
    }
  };

  const handleSubmit = async () => {
    if (!formData.termsAccepted) {
      alert('You must accept the terms and conditions to register.');
      return;
    }

    setIsSubmitting(true);
    try {
      const registration = await eventApiClient.createRegistration(
        eventCode!,
        formData
      );
      navigate(`/registration-confirmation/${registration.confirmationCode}`);
    } catch (error) {
      alert('Registration failed. Please try again.');
    } finally {
      setIsSubmitting(false);
    }
  };

  return (
    <PublicLayout>
      <div className="container mx-auto px-4 py-12 max-w-3xl">
        {/* Progress Indicator (2-step) */}
        <div className="mb-8">
          <div className="flex items-center justify-between">
            <span className={`text-sm ${currentStep === 1 ? 'text-blue-400 font-medium' : 'text-zinc-500'}`}>
              1. Your Details
            </span>
            <span className={`text-sm ${currentStep === 2 ? 'text-blue-400 font-medium' : 'text-zinc-500'}`}>
              2. Confirm Registration
            </span>
          </div>
          <div className="mt-2 h-2 bg-zinc-800 rounded-full overflow-hidden">
            <div
              className="h-full bg-blue-400 transition-all duration-300"
              style={{ width: `${(currentStep / 2) * 100}%` }}
            />
          </div>
        </div>

        {/* Steps */}
        {currentStep === 1 && (
          <PersonalDetailsStep formData={formData} setFormData={setFormData} />
        )}
        {currentStep === 2 && (
          <ConfirmRegistrationStep
            formData={formData}
            setFormData={setFormData}
            onEdit={() => setCurrentStep(1)}
          />
        )}

        {/* Navigation */}
        <div className="mt-8 flex justify-between">
          <Button variant="outline" onClick={handleCancel}>
            Cancel
          </Button>
          <div className="flex gap-4">
            {currentStep > 1 && (
              <Button variant="outline" onClick={handleBack}>
                Back
              </Button>
            )}
            {currentStep < 2 ? (
              <Button onClick={handleNext}>Next</Button>
            ) : (
              <Button onClick={handleSubmit} disabled={!formData.termsAccepted || isSubmitting}>
                {isSubmitting ? 'Submitting...' : 'Complete Registration'}
              </Button>
            )}
          </div>
        </div>
      </div>
    </PublicLayout>
  );
};
```

**SessionSelectionStep.tsx** (simplified):
```typescript
import { useQuery } from '@tanstack/react-query';
import { Checkbox } from '@/components/public/ui/checkbox';
import { eventApiClient } from '@/services/eventApiClient';

export const SessionSelectionStep = ({ eventId, formData, setFormData }) => {
  const { data: sessions, isLoading } = useQuery({
    queryKey: ['events', eventId, 'sessions'],
    queryFn: () => eventApiClient.getEventSessions(eventId),
  });

  const toggleSession = (sessionId: string) => {
    setFormData((prev) => ({
      ...prev,
      sessionIds: prev.sessionIds.includes(sessionId)
        ? prev.sessionIds.filter((id) => id !== sessionId)
        : [...prev.sessionIds, sessionId],
    }));
  };

  if (isLoading) return <div>Loading sessions...</div>;

  return (
    <div>
      <h2 className="text-2xl font-light mb-6">Select Sessions</h2>
      <div className="space-y-4">
        {sessions?.map((session) => (
          <div key={session.id} className="flex items-start gap-3 p-4 border border-zinc-800 rounded-lg">
            <Checkbox
              checked={formData.sessionIds.includes(session.id)}
              onCheckedChange={() => toggleSession(session.id)}
              disabled={session.registrationCount >= session.capacity}
            />
            <div className="flex-1">
              <h3 className="font-light">{session.title}</h3>
              <p className="text-sm text-zinc-400 mt-1">{session.startTime} - {session.endTime} • {session.room}</p>
              <p className="text-sm text-zinc-500 mt-2">
                {session.capacity - session.registrationCount > 0
                  ? `${session.capacity - session.registrationCount} spots remaining`
                  : 'FULL'}
              </p>
            </div>
          </div>
        ))}
      </div>
    </div>
  );
};
```
