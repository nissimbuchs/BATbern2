# Story 4.1.5: Registration Wizard (3-Step Flow)

## Status
Draft

## Story
**As a** public website visitor,
**I want** to register for the current event through an intuitive 3-step wizard,
**so that** I can select sessions and secure my spot at the event.

## Domain Context

### Primary Domain
Attendee Experience Domain (Event registration)

### Involved Services
- web-frontend (React 19 + TypeScript)
- Event Management Service (via API Gateway)
- API Gateway: `POST /api/v1/registrations`, `GET /api/v1/events/{id}/sessions`

### Cross-Domain Dependencies
- Event Management Service handles registration logic and capacity checks

## Requirements Context

### Related Functional Requirements
- Epic 4, Story 4.1
- AC 7-11: Registration wizard, form validation, session selection

## Acceptance Criteria

1. `/register/:eventId` route displays RegistrationWizard
2. **Step 1**: Personal info form (name, email, company, role) with validation
3. **Step 2**: Session selection with multi-select, shows capacity warnings
4. **Step 3**: Review & confirm with summary, terms acceptance
5. Progress indicator shows current step (1/3, 2/3, 3/3)
6. Navigation: [Back], [Next], [Cancel] buttons
7. Form state preserved when navigating between steps
8. **API Integration**: `POST /api/v1/registrations` on final submit
9. Session capacity checked in real-time via API
10. Validation errors displayed inline with clear messages

## Dev Notes - Implementation Guide

### Backend API Implementation

**RegistrationController.java** (Event Management Service):
```java
/**
 * Create registration (PUBLIC ACCESS - no auth required)
 * Allows anonymous public users to register for events
 */
@PostMapping("/registrations")
public ResponseEntity<RegistrationDTO> createRegistration(
    @RequestBody @Valid CreateRegistrationRequest request
) {
    Registration registration = registrationService.createRegistration(request);
    RegistrationDTO dto = registrationMapper.toDTO(registration);
    return ResponseEntity.status(HttpStatus.CREATED).body(dto);
}

/**
 * Get registration by ID (PUBLIC ACCESS)
 * Anyone with registration ID can view their registration
 */
@GetMapping("/registrations/{registrationId}")
public ResponseEntity<RegistrationDTO> getRegistration(
    @PathVariable String registrationId
) {
    Registration registration = registrationService.getRegistrationById(registrationId);
    return ResponseEntity.ok(registrationMapper.toDTO(registration));
}
```

### Component Implementation

**RegistrationWizard.tsx:**
```typescript
import { useState } from 'react';
import { useParams, useNavigate } from 'react-router-dom';
import { PublicLayout } from '@/components/public/PublicLayout';
import { PersonalInfoStep } from '@/components/public/Registration/PersonalInfoStep';
import { SessionSelectionStep } from '@/components/public/Registration/SessionSelectionStep';
import { ReviewStep } from '@/components/public/Registration/ReviewStep';
import { Button } from '@/components/public/ui/button';
import { eventApiClient } from '@/services/eventApiClient';

export const RegistrationWizard = () => {
  const { eventId } = useParams();
  const navigate = useNavigate();
  const [currentStep, setCurrentStep] = useState(1);
  const [formData, setFormData] = useState({
    firstName: '',
    lastName: '',
    email: '',
    company: '',
    role: '',
    sessionIds: [] as string[],
    termsAccepted: false,
  });
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleNext = () => setCurrentStep((prev) => Math.min(prev + 1, 3));
  const handleBack = () => setCurrentStep((prev) => Math.max(prev - 1, 1));
  const handleCancel = () => {
    if (confirm('Are you sure you want to cancel registration?')) {
      navigate('/current-event');
    }
  };

  const handleSubmit = async () => {
    setIsSubmitting(true);
    try {
      const registration = await eventApiClient.createRegistration({
        eventId: eventId!,
        ...formData,
      });
      navigate(`/registration-confirmation/${registration.id}`);
    } catch (error) {
      alert('Registration failed. Please try again.');
    } finally {
      setIsSubmitting(false);
    }
  };

  return (
    <PublicLayout>
      <div className="container mx-auto px-4 py-12 max-w-3xl">
        {/* Progress Indicator */}
        <div className="mb-8">
          <div className="flex items-center justify-between">
            <span className={`text-sm ${currentStep === 1 ? 'text-blue-400 font-medium' : 'text-zinc-500'}`}>
              1. Personal Info
            </span>
            <span className={`text-sm ${currentStep === 2 ? 'text-blue-400 font-medium' : 'text-zinc-500'}`}>
              2. Select Sessions
            </span>
            <span className={`text-sm ${currentStep === 3 ? 'text-blue-400 font-medium' : 'text-zinc-500'}`}>
              3. Review
            </span>
          </div>
          <div className="mt-2 h-2 bg-zinc-800 rounded-full overflow-hidden">
            <div
              className="h-full bg-blue-400 transition-all duration-300"
              style={{ width: `${(currentStep / 3) * 100}%` }}
            />
          </div>
        </div>

        {/* Steps */}
        {currentStep === 1 && (
          <PersonalInfoStep formData={formData} setFormData={setFormData} />
        )}
        {currentStep === 2 && (
          <SessionSelectionStep
            eventId={eventId!}
            formData={formData}
            setFormData={setFormData}
          />
        )}
        {currentStep === 3 && (
          <ReviewStep
            formData={formData}
            setFormData={setFormData}
            onEdit={(step) => setCurrentStep(step)}
          />
        )}

        {/* Navigation */}
        <div className="mt-8 flex justify-between">
          <Button variant="outline" onClick={handleCancel}>
            Cancel
          </Button>
          <div className="flex gap-4">
            {currentStep > 1 && (
              <Button variant="outline" onClick={handleBack}>
                Back
              </Button>
            )}
            {currentStep < 3 ? (
              <Button onClick={handleNext}>Next</Button>
            ) : (
              <Button onClick={handleSubmit} disabled={!formData.termsAccepted || isSubmitting}>
                {isSubmitting ? 'Submitting...' : 'Complete Registration'}
              </Button>
            )}
          </div>
        </div>
      </div>
    </PublicLayout>
  );
};
```

**PersonalInfoStep.tsx** (simplified):
```typescript
import { useForm } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { z } from 'zod';
import { Form, FormControl, FormField, FormItem, FormLabel, FormMessage } from '@/components/public/ui/form';
import { Input } from '@/components/public/ui/input';

const schema = z.object({
  firstName: z.string().min(1, 'First name is required'),
  lastName: z.string().min(1, 'Last name is required'),
  email: z.string().email('Invalid email address'),
  company: z.string().min(1, 'Company is required'),
  role: z.string().min(1, 'Role is required'),
});

export const PersonalInfoStep = ({ formData, setFormData }) => {
  const form = useForm({
    resolver: zodResolver(schema),
    defaultValues: formData,
  });

  const onUpdate = (data: any) => {
    setFormData((prev) => ({ ...prev, ...data }));
  };

  return (
    <div>
      <h2 className="text-2xl font-light mb-6">Personal Information</h2>
      <Form {...form}>
        <form onBlur={form.handleSubmit(onUpdate)} className="space-y-4">
          <FormField name="firstName" control={form.control} render={({ field }) => (
            <FormItem>
              <FormLabel>First Name</FormLabel>
              <FormControl><Input {...field} /></FormControl>
              <FormMessage />
            </FormItem>
          )} />
          {/* Similar for lastName, email, company, role */}
        </form>
      </Form>
    </div>
  );
};
```

**SessionSelectionStep.tsx** (simplified):
```typescript
import { useQuery } from '@tanstack/react-query';
import { Checkbox } from '@/components/public/ui/checkbox';
import { eventApiClient } from '@/services/eventApiClient';

export const SessionSelectionStep = ({ eventId, formData, setFormData }) => {
  const { data: sessions, isLoading } = useQuery({
    queryKey: ['events', eventId, 'sessions'],
    queryFn: () => eventApiClient.getEventSessions(eventId),
  });

  const toggleSession = (sessionId: string) => {
    setFormData((prev) => ({
      ...prev,
      sessionIds: prev.sessionIds.includes(sessionId)
        ? prev.sessionIds.filter((id) => id !== sessionId)
        : [...prev.sessionIds, sessionId],
    }));
  };

  if (isLoading) return <div>Loading sessions...</div>;

  return (
    <div>
      <h2 className="text-2xl font-light mb-6">Select Sessions</h2>
      <div className="space-y-4">
        {sessions?.map((session) => (
          <div key={session.id} className="flex items-start gap-3 p-4 border border-zinc-800 rounded-lg">
            <Checkbox
              checked={formData.sessionIds.includes(session.id)}
              onCheckedChange={() => toggleSession(session.id)}
              disabled={session.registrationCount >= session.capacity}
            />
            <div className="flex-1">
              <h3 className="font-light">{session.title}</h3>
              <p className="text-sm text-zinc-400 mt-1">{session.startTime} - {session.endTime} • {session.room}</p>
              <p className="text-sm text-zinc-500 mt-2">
                {session.capacity - session.registrationCount > 0
                  ? `${session.capacity - session.registrationCount} spots remaining`
                  : 'FULL'}
              </p>
            </div>
          </div>
        ))}
      </div>
    </div>
  );
};
```

import { Checkbox } from '@/components/public/ui/checkbox';

export const ReviewStep = ({ formData, setFormData, onEdit }) => {
  return (
    <div>
      <h2 className="text-2xl font-light mb-6">Review & Confirm</h2>

      {/* Personal Info Summary */}

        <div className="flex items-center justify-between mb-2">
          <h3 className="font-light">Personal Information</h3>

        </div>

          {formData.email}<br />

        </p>
      </div>

      {/* Sessions Summary */}
      <div className="mb-6 p-4 border border-zinc-800 rounded-lg">
        <div className="flex items-center justify-between mb-2">

          </button>
        </div>
      </div>

      {/* Terms */}
      <div className="flex items-start gap-3">
        <Checkbox
          checked={formData.termsAccepted}
          onCheckedChange={(checked) =>
            setFormData((prev) => ({ ...prev, termsAccepted: !!checked }))
          }
        />
        <label className="text-sm text-zinc-400">
          I accept the terms and conditions and privacy policy
        </label>
      </div>
    </div>
  );
};
