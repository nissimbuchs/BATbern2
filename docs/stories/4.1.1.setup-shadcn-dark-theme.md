### Installation Commands

**Step 1: Install dependencies**
```bash
cd web-frontend
npm install @radix-ui/react-dialog @radix-ui/react-dropdown-menu @radix-ui/react-select @radix-ui/react-tabs @radix-ui/react-slot
npm install class-variance-authority clsx tailwind-merge
```

**Step 2: Initialize shadcn/ui**
```bash
npx shadcn-ui@latest init
# Select:
# - TypeScript: Yes
# - Style: New York
# - Base color: Zinc
# - CSS variables: Yes
# - Import alias: @/components/public/ui
```

**Step 3: Install components**
```bash
npx shadcn-ui@latest add button card badge dialog form tabs
```

### Configuration Files

**components.json** (created by init):
```json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "new-york",
  "rsc": false,
  "tsx": true,
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/index.css",
    "baseColor": "zinc",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components/public/ui",
    "utils": "@/lib/utils"
  }
}
```

**src/lib/utils.ts** (utility for component composition):
```typescript
import { type ClassValue, clsx } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```

**tailwind.config.js** (update for dark theme):
```javascript
/** @type {import('tailwindcss').Config} */
export default {
  darkMode: ["class"],
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {
      colors: {
        // Existing MUI colors remain
        // shadcn zinc palette (auto-configured by CLI)
        // Custom accents
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
      },
    },
  },
  plugins: [require("tailwindcss-animate")],
}
```

**src/index.css** (ensure Tailwind directives present):
```css
@tailwind base;
@tailwind components;
@tailwind utilities;

/* Dark theme CSS variables will be auto-injected by shadcn */
```
